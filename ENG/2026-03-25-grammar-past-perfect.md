# Study Note: 과거완료 시제 (Past Perfect)

## 목표
- 과거 시제(Simple Past)와 과거완료(Past Perfect)의 차이를 이해하고 정확히 사용하기

## 배운 내용
- had + p.p. = "과거의 과거" — 두 과거 사건 중 먼저 일어난 일에 사용
- 시그널 단어: by the time, before, when, because, already
- "so(그래서)" 앞에 오는 일 = 원인 = 먼저 일어난 일 = had + p.p.

## 예문
- When I **arrived** at the station, the train **had already left**. (기차 떠남 → 내가 도착)
- She **didn't recognize** him because he **had changed** so much. (그가 변함 → 못 알아봄)
- By the time we **ordered** food, we **had waited** for 30 minutes. (기다림 → 주문)
- Before the movie **began**, we **had already eaten** all the popcorn. (팝콘 먹음 → 영화 시작)
- I **was** tired because I **had worked** all night. (밤새 일함 → 피곤함)

## 내가 헷갈린 점
- 시그널 단어(by the time, before) 있으면 순서 판단 OK
- 시그널 없이 so, because만 있을 때 과거완료 방향을 반대로 넣음 (ate / had invited → had eaten / invited)
- 불규칙 동사 형태 실수: hadn't sleep → hadn't **slept**, movie begin → movie **began**
- "밤새" 표현: along the night ✗ → **all night** ✓

## 복습 포인트
- 시그널 단어 없는 문장에서 "뭐가 먼저 일어났는가?" 판단 연습
- 불규칙 동사 변화: sleep-slept-slept, begin-began-begun
- 영어 시간 표현: all night, all day, all morning
