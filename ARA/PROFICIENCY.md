## 영역별 숙련도

### 어휘 (Vocabulary)
- **레벨**: S0
- **추정 어휘 수**: 0~10개
- **강점**: 기초 인사 표현을 막 접하기 시작한 단계
- **약점**: 실사용 어휘가 거의 없고, 문자와 소리를 바로 연결하기 어려움

### 문법 (Grammar)
- **레벨**: S0
- **이해하는 문법**: 문장 구조 학습 전, 가장 기본적인 인사 표현만 접한 상태
- **취약한 문법**: 성 구분, 어순, 동사 활용 등 전반적인 문법 체계 전부

### 읽기 (Reading)
- **레벨**: S0
- **읽을 수 있는 수준**: 일부 문자 인식 전 단계, 단어 읽기 불가
- **어려운 부분**: 문자 모양 구분, 모음 표기, 오른쪽에서 왼쪽으로 읽기

### 쓰기 (Writing)
- **레벨**: S0
- **가능한 수준**: 문자 쓰기 연습 전 단계
- **어려운 부분**: 자모 형태 변화, 획순, 소리와 철자 연결

## 레벨 기준

| 레벨 | 설명 |
|------|------|
| **S0 - 입문** | 알파벳/문자 체계를 배우는 단계, 거의 아는 것이 없음 |
| **S1 - 초급** | 기초 인사·숫자·단순 문장 가능, 일상 단어 100개 이내 |
| **S2 - 초중급** | 간단한 일상 대화 가능, 기초 문법 이해, 어휘 300~500개 |
| **S3 - 중급** | 일상 주제 대화 가능, 주요 문법 활용, 어휘 1000개 이상 |
| **S4 - 중상급** | 복잡한 주제 토론 가능, 뉴스·기사 이해, 어휘 3000개 이상 |
| **S5 - 상급** | 전문 분야 의사소통 가능, 원어민에 가까운 이해력 |
