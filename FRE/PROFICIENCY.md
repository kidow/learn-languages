## 영역별 숙련도

### 어휘 (Vocabulary)
- **레벨**: S0
- **추정 어휘 수**: 0~10개
- **강점**: 몇몇 인사 표현을 들어본 적은 있는 수준
- **약점**: 실사용 어휘가 거의 없고, 철자와 발음의 차이에 익숙하지 않음

### 문법 (Grammar)
- **레벨**: S0
- **이해하는 문법**: 문법 학습 전, 고정 표현을 따라 말하는 단계
- **취약한 문법**: 관사, 성수 일치, 동사 변화, 부정문 등 전반

### 읽기 (Reading)
- **레벨**: S0
- **읽을 수 있는 수준**: 짧은 단어도 독립적으로 읽기 어려움
- **어려운 부분**: 묵음 처리, 연음, 악센트 기호와 발음 규칙

### 쓰기 (Writing)
- **레벨**: S0
- **가능한 수준**: 자기소개 문장 작성 전 단계
- **어려운 부분**: 철자 정확도, 악센트 기호 사용, 기본 문장 구성

## 레벨 기준

| 레벨 | 설명 |
|------|------|
| **S0 - 입문** | 알파벳/문자 체계를 배우는 단계, 거의 아는 것이 없음 |
| **S1 - 초급** | 기초 인사·숫자·단순 문장 가능, 일상 단어 100개 이내 |
| **S2 - 초중급** | 간단한 일상 대화 가능, 기초 문법 이해, 어휘 300~500개 |
| **S3 - 중급** | 일상 주제 대화 가능, 주요 문법 활용, 어휘 1000개 이상 |
| **S4 - 중상급** | 복잡한 주제 토론 가능, 뉴스·기사 이해, 어휘 3000개 이상 |
| **S5 - 상급** | 전문 분야 의사소통 가능, 원어민에 가까운 이해력 |
