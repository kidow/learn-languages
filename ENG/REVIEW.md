# English Review

## 자주 틀리는 표현

### 동사 형태
- hadn't **sleep** ✗ → hadn't **slept** ✓ (sleep-slept-slept 불규칙)
- the movie **begin** ✗ → the movie **began** ✓ (begin-began-begun 불규칙)

### 전치사/부사
- worked **along the night** ✗ → worked **all night** ✓ ("밤새"는 all night)

### 콩글리시
- **gym center** ✗ → **gym** ✓
- **play the swimming** ✗ → **go swimming** ✓
- **take the vibe coding** ✗ → **do some vibe coding** ✓

## 다시 볼 문법

### 과거완료 (had + p.p.) — 2026-03-25 학습
- "과거의 과거" = 먼저 일어난 일에 had + p.p.
- 시그널 단어: by the time, before, when, because, already
- **주의:** 시그널 단어 없이 문맥만으로 판단할 때 순서 헷갈림 → "so 앞 = 원인 = 먼저 일어난 일 = had + p.p."

### 관계대명사 (which vs that vs what) — 아직 미학습
- 레벨 테스트에서 which→what, which→that 오답
- 비제한적 관계절 (콤마 + which) 구분 필요

### 도치구문 — 아직 미학습
- Not only **did she pass** (도치) vs Not only **she passed** (오답)

### wish + 과거완료 — 아직 미학습
- I wish I **had studied** ✓ vs I wish I **study** ✗

## 복습 우선순위
1. 과거완료 시제 — 시그널 없는 문맥 판단 연습 (학습 완료, 복습 필요)
2. 관계대명사 which/that/what — 다음 세션 예정
3. 도치구문 — Phase 1 후반
