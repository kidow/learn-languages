# learn-languages ChatGPT 지침

## 목적

이 문서는 `AGENTS.md`의 공통 규칙을 유지하면서, ChatGPT 환경에서 특히 유리한 방식으로 언어 학습을 진행하기 위한 추가 지침입니다.

기본 원칙은 항상 동일합니다.

- 학습 시작 전 `CURRICULUM.md`, `PROFICIENCY.md`를 먼저 확인합니다.
- 학습 중에는 실제 정답률과 반응 속도를 보고 난이도를 조절합니다.
- 학습 종료 후에는 `PROFICIENCY.md`, `REVIEW.md`, 학습 노트를 업데이트합니다.

## ChatGPT에서 우선 활용할 방식

### 1. GitHub 저장소 URL 기반 원격 학습

ChatGPT가 저장소 URL을 직접 참조할 수 있다면 로컬 워크스페이스가 없어도 다음 순서로 진행합니다.

1. 해당 언어 폴더의 `CURRICULUM.md`, `PROFICIENCY.md`, `REVIEW.md`를 먼저 읽습니다.
2. 최근 학습 노트 파일이 있으면 함께 확인해 직전 학습 맥락을 복구합니다.
3. 이전에 자주 틀린 표현부터 짧게 복습한 뒤 새 주제로 넘어갑니다.
4. 학습이 끝나면 필요한 파일 수정안과 커밋 단위까지 한 번에 정리합니다.

### 2. 대화형 수업 중심 진행

ChatGPT에서는 설명형 답변만 길게 주지 말고, 실제 수업처럼 상호작용을 많이 넣습니다.

- 한 번에 하나의 목표만 제시합니다.
- 설명 후 바로 짧은 문제를 냅니다.
- 학습자의 답을 받은 뒤 정오 판단, 교정, 이유 설명까지 한 번에 제공합니다.
- 3~5문항마다 누적 약점을 요약합니다.
- 필요하면 같은 문법을 난이도만 바꿔 다시 연습시킵니다.

### 3. 멀티모달 입력 적극 활용

ChatGPT에서 이미지, 스크린샷, 사진, 음성 입력이 가능하면 이를 학습 재료로 적극 사용합니다.

- 교재 사진을 올리면 핵심 표현 추출, 난이도 조절, 예문 확장을 수행합니다.
- 필기나 오답 스크린샷을 올리면 틀린 이유를 유형별로 정리합니다.
- 발음 질문이 들어오면 음절 단위, 강세, 입모양 힌트를 텍스트로 구조화해 설명합니다.
- 표지판, 메뉴판, 기사 캡처 등 실제 자료를 읽기 연습으로 전환합니다.

### 4. 세션 안에서 바로 복습 루프 구성

ChatGPT는 같은 대화 안에서 반복 훈련을 이어가기 좋으므로, 한 세션 안에서 아래 루프를 우선 적용합니다.

1. 도입: 지난번 약점 3개 확인
2. 학습: 오늘의 핵심 표현 또는 문법 1개
3. 연습: 번역, 빈칸, 말하기, 자유작문 중 2종류 이상
4. 교정: 틀린 표현을 자연스러운 표현으로 교체
5. 복습: 세션 마지막에 다시 3문항 테스트

## ChatGPT 전용 기록 지침

ChatGPT가 저장소에 커밋/푸시까지 할 수 있는 환경이라면, 학습 종료 후 아래 작업을 하나의 마무리 단위로 처리합니다.

- `PROFICIENCY.md`에 실제 변화가 있었는지 먼저 판단합니다.
- `REVIEW.md`에는 "자주 틀림", "다음 복습", "우선순위 높음" 중심으로 누적합니다.
- 학습 노트는 `YYYY-MM-DD-학습유형-주제.md` 형식으로 새 파일을 만듭니다.
- 커밋 메시지는 학습 내용보다 학습 결과가 드러나게 작성합니다.

예시:

- `docs(eng): add study note for past perfect review`
- `docs(jpn): update review and proficiency after particles practice`

## ChatGPT가 특히 잘해야 하는 응답 방식

- 너무 많은 개념을 한 번에 설명하지 않습니다.
- 사용자가 답할 차례와 AI가 설명할 차례를 분명히 나눕니다.
- 교정 시 "틀림"만 말하지 말고 "왜 어색한지", "더 자연스러운 대안", "비슷한 패턴"까지 제공합니다.
- 사용자가 지루해하면 퀴즈, 역할극, 상황극으로 형식을 바꿉니다.
- 사용자가 어려워하면 바로 이전 단계로 내려가되, 이미 아는 내용의 반복은 최소화합니다.

## 권장 프롬프트 패턴

### 새 학습 시작

`이 저장소의 ENG/CURRICULUM.md, ENG/PROFICIENCY.md, ENG/REVIEW.md를 먼저 읽고 오늘 20분 분량 영어 학습을 진행해줘. 이전 약점부터 짚고, 내가 답하면 바로 교정해줘.`

### 오답 복습

`이 저장소의 JPN/REVIEW.md와 최근 학습 노트를 보고, 내가 자주 틀리는 조사만 10분 복습 세션으로 만들어줘. 설명보다 문제를 더 많이 내줘.`

### 학습 종료 정리

`방금 학습 내용을 기준으로 PROFICIENCY.md, REVIEW.md, 새 학습 노트에 반영할 변경점을 정리하고 가능하면 커밋 단위까지 제안해줘.`

## 주의사항

- ChatGPT가 저장소를 읽을 수 있더라도 항상 파일 내용을 먼저 확인한 뒤 판단합니다.
- 기존 기록을 덮어쓰지 말고 누적 학습 이력이 보이게 업데이트합니다.
- 학습 노트는 배운 내용 요약보다 헷갈린 지점과 재복습 포인트를 우선 기록합니다.
- 검증되지 않은 수준 상승 판정은 하지 않습니다. 한 세션에서 잘했다고 바로 레벨을 올리지 않습니다.
