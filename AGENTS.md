# learn-languages 프로젝트 지침

## 프로젝트 개요

외국어 학습 내용을 Markdown으로 기록·복습하는 저장소입니다.

## 언어 학습 시 필수 워크플로우

### 1. 학습 시작 전: CURRICULUM.md와 PROFICIENCY.md 확인

학습을 시작할 때 **반드시** 해당 언어의 `{언어폴더}/CURRICULUM.md`와 `{언어폴더}/PROFICIENCY.md`를 먼저 읽으세요.

- `CURRICULUM.md`: 해당 언어에서 어떤 순서로 무엇을 가르칠지, 현재 단계에서 우선 다룰 주제를 파악합니다.
- `PROFICIENCY.md`: 현재 숙련도와 강점·약점을 확인해서 난이도를 조절합니다.

- S0~S1: 기초 단어, 인사, 문자 체계 위주
- S2: 일상 회화, 기초 문법 패턴
- S3: 주제별 대화, 문법 심화
- S4~S5: 뉴스·기사 독해, 복잡한 토론, 작문

### 2. 학습 중: 수준 조절

- 학습자가 너무 쉬워하면 난이도를 올리고, 어려워하면 낮추세요.
- 학습자의 답변 정확도를 관찰해서 실제 수준을 지속적으로 판단하세요.
- 틀린 부분은 즉시 교정하되, 왜 틀렸는지 설명을 포함하세요.

### 3. 학습 완료 후: 기록 업데이트

학습 세션이 끝나면 아래 파일들을 업데이트하세요:

#### PROFICIENCY.md 업데이트
- 숙련도에 변화가 있으면 해당 영역의 레벨·강점·약점을 수정합니다.
- `최근 업데이트` 날짜를 갱신합니다.
- `학습 이력` 테이블에 변경 내역을 추가합니다.
- 변화가 없더라도 약점·강점 정보가 더 구체화되면 업데이트합니다.

#### REVIEW.md 업데이트
- 자주 틀리는 표현, 다시 볼 문법, 복습 우선순위를 누적 정리합니다.

#### 학습 노트 생성
- `YYYY-MM-DD-학습유형-주제.md` 형식으로 생성합니다.
- `TEMPLATE/study-note-template.md`의 구조를 따릅니다.
- 한 파일에 한 주제만 기록합니다.

## 파일 구조 규칙

```
{언어폴더}/
├─ CURRICULUM.md     # 학습 순서·단계별 목표 (필수)
├─ PROFICIENCY.md    # 숙련도 현황 (필수)
├─ REVIEW.md         # 복습 포인트 누적 (필수)
└─ YYYY-MM-DD-*.md   # 학습 노트 (세션마다 생성)
```

## 레벨 체계 (S0~S5)

| 레벨 | 설명 |
|------|------|
| S0 | 알파벳/문자 체계 학습 단계 |
| S1 | 기초 인사·숫자·단순 문장, 어휘 ~100개 |
| S2 | 간단한 일상 대화, 기초 문법, 어휘 300~500개 |
| S3 | 일상 주제 대화, 주요 문법 활용, 어휘 1000개+ |
| S4 | 복잡한 주제 토론, 뉴스·기사 이해, 어휘 3000개+ |
| S5 | 전문 분야 의사소통, 원어민 수준 이해력 |

## 학습 노트 작성 원칙

- 배운 내용 자체보다 **헷갈린 표현**과 **다시 볼 포인트**를 더 중요하게 기록합니다.
- 하루에 주제가 여러 개면 파일도 여러 개로 나눕니다.
- 복습 가능한 기록을 만드는 것이 목표입니다.
